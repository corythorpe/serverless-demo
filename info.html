The Modern Developer's Journey

The serverless paradigm transforms the developer workflow into a streamlined, five-step process focused on rapid integration and iteration, as shown below. This contrasts sharply with traditional methods that involve extensive infrastructure planning and setup before any code is written.

1. Select Model

Choose a pre-trained model from a catalog or use a custom-trained one.

2. Generate API Key

Obtain an API key to access pre-available models instantly.

3. Utilize Shared Endpoint

Access the common API endpoint; your key grants individual access.

4. Interact with Model via API

Make a `POST` request to the endpoint using your API key in the authorization header to send data and receive model responses.

5. Test & Iterate

Launch, test with real users, and iterate rapidly based on feedback and performance.
